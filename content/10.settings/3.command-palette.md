Command Palette gives you one searchable command surface for navigation, creation, settings, and desktop actions.

Open it with **Cmd/Ctrl + K**.

## Available Across Msty Studio

The Command Palette remains available while you are working in navigable hubs and dialogs, including:

- Model Hub
- Settings
- Toolbox
- Persona Studio
- Prompt Studio
- Turnstiles
- Knowledge Stacks
- Environments
- Notifications, Insights, and the Log Viewer

Press **Cmd/Ctrl + K** without manually closing the current hub. When you run a navigation command, Msty closes the existing navigable surface and opens the selected destination.

The palette may remain unavailable while a blocking confirmation, editor drawer, license prompt, or other modal action requires your attention.

## Major Parts

### Create commands
Start new conversations, split chats, persona or crew chats, forge canvases, and workspaces.

### Navigation commands
Jump directly between Model Hub, Toolbox, Persona Studio, Prompt Studio, Turnstiles, Knowledge Stacks, environments, notifications, and other major surfaces.

### Settings and appearance commands
Open settings sections, toggle mini map, and quickly switch theme, mode, split width, accent, or base color.

### Desktop commands
On Desktop, run actions like checking for updates or opening the log viewer from the same interface.

## Using Command Palette

1. Press **Cmd/Ctrl + K**.
2. Type what you want to do, such as `new`, `persona`, or `settings`.
3. Move through results with arrow keys or `Tab`.
4. Press `Enter` to run the selected command.
5. Press `Esc` to close.
