Msty Studio is currently available exclusively to **Aurum** license holders.

These onboarding instructions are meant for new Msty users. If you are an existing Msty App user, check out the [Quick Start](/getting-started/quick-start) guide.

## Onboarding Instructions

::steps

1. [Purchase an Aurum license](https://msty.app/pricing) if you haven't already and keep your license key handy

2. Open a browser and navigate to Msty Studio at [https://msty.studio](https://msty.studio)

3. Enter your license key and click **Next**

4. You'll now have a few options on how you can proceed. In order to make use of Msty, you will need to add either a local or remote model provider.
   ![Msty Studio Initial Setup](/images/initial-setup.png)
   - To add a Remote Provider, click **Add Provider**. There are many available options, such as OpenAI, Claude, and Gemini. You will first need an API Key from the desired provider in order to add the remote provider to Msty.
   - To add a Local Provider, select the **Get Sidecar** option. This may be the better option to begin with at this time, as Sidecar unlocks many additional features in Msty Studio - and, you can always add new Remote Providers later. Installing Sidecar will take you through a Local AI Setup flow - check out the [Sidecar](/getting-started/sidecar) documentation for additional info.
5. Once you have your Remote or Local Provider set up, click on **Let's Go!**
