**Msty Studio Desktop Alpha is now available for Aurum license holders!**

All of the features you love in Msty Studio Web are now available in a desktop application. 

Check out the our [announcement blog post](https://msty.ai/blog/announcing-studio-desktop) for more details.

Please join our [Discord](https://msty.app/discord) to share your feedback and help us improve the desktop experience. 

![msty studio](/images/image-comparison.png)

### Quick Facts
1. Alpha available for Aurum Lincense holders only. 
2. Studio Desktop installs separately from Msty App (v1.0)
3. You do not need remove Msty App (v1.0), you can continue using it while using Studio Desktop.
4. You will need your Aurum license key to activate Studio Desktop - see [Manage License](/how-tos/manage-license) for details on how to get your license key.
5. Studio Desktop is available for Windows, Mac, and Linux.
6. A migration tool is being developed to help you migrate your existing Msty App (v1.0) data to Studio Desktop.
