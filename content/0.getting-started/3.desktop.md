**Msty Studio Desktop Alpha is now available for Aurum license holders!**

All of the features you love in Msty Studio Web are now available in a desktop application. 

Check out the our [announcement blog post](https://msty.ai/blog/announcing-studio-desktop) for more details.

Please join our [Discord](https://msty.app/discord) to share your feedback and help us improve the desktop experience. 

![msty studio](/images/image-comparison.png)

### Quick Facts
1. Alpha available for Aurum License holders only.
2. Studio Desktop installs separately from Msty App (v1.x).
3. You do not need remove Msty App (v1.x), you can continue using it while using Studio Desktop.
4. You will need your Aurum license key to activate Studio Desktop - see [Manage License](/how-tos/manage-license) for details on how to get your license key.
5. Studio Desktop activation will count against your Aurum activated devices limit, so you may need to deactivate a device if you have reached your limit.
6. Studio Desktop is available for Windows, Mac, and Linux.
7. A migration tool is being developed to help you migrate your existing Msty App (v1.x) data to Studio Desktop.
8. Studio Desktop does not require Msty Sidecar to use tools, RTDs, and local models. Sidecar is a companion for the web version of Msty Studio. However, you can optionally connect Studio Web to Studio Desktop instead of Sidecar to use tools, RTDs, and local models.

### Linux

Linux requires some additional manual steps after downloading. 

---

### AppImage Instructions

[Download Msty Studio Desktop (x64 AppImage)](https://next-assets.msty.studio/app/alpha/linux/MstyStudio_x86_64.AppImage)

After download is complete, run the following command in your terminal to make the AppImage executable:

```bash
chmod a+x ./<DownloadedFileName>.AppImage
```

Then, you can run the AppImage file:

```bash
./<DownloadedFileName>.AppImage --no-sandbox

```
Note: You may need to install appropriate packages like libfuse2 (if your system does not have them already) in order to run the AppImage file.

---

### Deb Instructions

[Download Msty Studio Desktop (x64 deb)](https://next-assets.msty.studio/app/alpha/linux/MstyStudio_amd64.deb)

After download is complete, run the following command to install the deb package:
```
sudo apt install ./<DownloadedFileName>.deb
```

Once the installation is complete, run the following command to launch Msty Studio Desktop:

```bash
msty
```

Sometimes the installation may experience permission issues if the '_apt' user does not have enough privileges. In that case, run the following two commands to mitigate this issue and then run Msty again.

```bash
sudo chown root:root /opt/Msty/chrome-sandbox
```
```bash
sudo chmod 4755 /opt/Msty/chrome-sandbox
```
