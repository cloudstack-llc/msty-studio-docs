Msty Claw supports Discord, Telegram, and WhatsApp channels.

Use this as the high-level setup flow. For full platform-specific requirements, always follow each platform's official docs.

## Discord

In Msty Claw:

1. Open **Settings > Channels**.
2. Add a channel with platform set to **Discord**.
3. Paste your Discord Bot Token.
4. Connect the channel, then set your chat target using the Discord channel ID.

Official docs:

- [Discord Developer Quick Start](https://docs.discord.com/developers/quick-start/getting-started){target="_blank"}

## Telegram

In Msty Claw:

1. Open **Settings > Channels**.
2. Add a channel with platform set to **Telegram**.
3. Create a bot using `@BotFather` and paste the bot token.
4. Set the target chat ID for the bot.

Official docs:

- [Telegram Bots FAQ](https://core.telegram.org/bots/faq){target="_blank"}

## WhatsApp

In Msty Claw:

1. Open **Settings > Channels**.
2. Add a channel with platform set to **WhatsApp**.
3. Connect and scan the QR code from your phone.
4. Send a message from the chat you want to use, then choose or paste the chat JID.

Official docs:

- [WhatsApp Help Center](https://faq.whatsapp.com){target="_blank"}
- [WhatsApp Business Platform Docs](https://developers.facebook.com/docs/whatsapp){target="_blank"}

## Integration tips

- Start with one channel and one bot until your workflow is stable.
- Keep bot tokens and keys private.
- Test in a private channel/chat before inviting others.
