MstyClaw is an OpenClaw-style desktop agent app from Msty. It is designed for broader agent workflows, not just dedicated AI coding tasks, and it is currently separate from Msty Studio.

Download MstyClaw from [msty.ai/claw](https://msty.ai/claw){target="_blank"}.

::alert{type="info" icon="tabler:info-circle"}
MstyClaw currently supports Apple Silicon macOS. Windows and Linux support are coming soon.
::

::alert{type="warning" icon="tabler:alert-circle"}
MstyClaw is currently in open beta and is free for personal use only.
::

## MstyClaw vs Msty Studio

MstyClaw is **not** Msty Studio. They are separate products today.

Over time, the products will become more closely integrated, but right now you should treat MstyClaw as its own app with its own setup and settings.

## What MstyClaw can do

- Run broader agent workflows with local or cloud models
- Connect chat channels like Discord, Telegram, and WhatsApp
- Run bots in a managed workspace or an existing project folder
- Run bots on host runtime or container runtime
- Use tools via MCP servers and custom skills

For product updates, examples, and the latest release information, check [msty.ai/claw](https://msty.ai/claw){target="_blank"}.

## Workspace and container options

When you create a bot, MstyClaw lets you choose:

- **Managed workspace**: an isolated workspace folder created for that bot
- **Project folder**: a specific folder you choose on disk
- **Host runtime**: runs directly on your machine
- **Container runtime**: runs in Docker or Podman with optional network access
