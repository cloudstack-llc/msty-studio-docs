This page covers the general setup flow.

## App setup (recommended)

::steps
### Install Msty Claw
Download it from [msty.ai/claw](https://msty.ai/claw){target="_blank"} and launch the app.

### Connect a model provider
During onboarding (or later in **Settings > Providers**), connect a provider and pick a model.

### Connect at least one channel
In **Settings > Channels**, add Discord, Telegram, or WhatsApp and connect it.

### Create your first bot
Choose workspace mode (managed or project folder), runtime (host or container), and access controls.
::

## Notes

- Msty Claw is currently in open beta and free for personal use only.
- Container runtime requires Docker or Podman.
- If container network is blocked, web access is also blocked.
- Bots require a connected channel and chat target.
