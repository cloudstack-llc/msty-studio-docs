Finding the right icon for your projects is now easier with AI-powered suggestions. By connecting an AI model to the Icon Finder, you can get intelligent recommendations based on simple text prompts.

## Setting Up AI Icon Assistance

::steps
### Open Model Hub
Navigate to **Model Hub** in the application.

### Configure Icon Finder
Under **Model Squad**, select **Icon Finder** from the list.

### Assign a Model
Choose and assign a model to power the icon finding feature.

### Use AI Suggestions
When editing a project (clicking ellipses > Edit Project > Icon dropdown), you'll see a new **AI Prompt** section. Type a keyword like "fun" and the AI will recommend matching icons from the available set.
::
