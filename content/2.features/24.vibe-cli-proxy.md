Integrate models from CLI tools directly into Msty Studio. This integration supports multiple CLI providers and makes their models available within your conversations.

### Supported CLI Providers

- Claude Code
- Codex
- Google Gemini
- iFlow
- Github Copilot
- Antigravity

### Link CLI Provider

::steps
### Open CLI Settings
Go to **Settings > Vibe CLI Proxy**.

### Authorize
Under Authentication, select your provider and click **Authorize**.

### Complete Flow
Follow the authorization prompts to connect.
::

### Use CLI Models

::steps
### Select Model
In any conversation, open the model selector.

### Choose Provider
Models from authorized CLI providers appear in the list. Select one to begin chatting.
::

::alert{type="info" icon="tabler:info-circle"}
CLI models work alongside your other configured models—no extra setup needed after authorization.
::
